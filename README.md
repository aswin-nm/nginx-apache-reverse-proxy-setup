
# 🔁 Nginx + Apache Reverse Proxy Setup

## 📌 Architecture
Internet → Nginx (Port 80) → Apache (Port 8080)

## 🛠️ Tools Used
- Ubuntu 22.04
- Nginx (Reverse Proxy)
- Apache2 (Backend Server)

## 📋 Steps
1. Install Nginx and Apache
2. Change Apache port to 8080
3. Configure Nginx proxy_pass to Apache
4. Test with browser and curl
